Enables Cluster Maps on all pages.

== Description ==

This plugin adds the required javascript for Clustr Maps.

For more information visit:

[Clustr Maps](http://www.clustrmaps.com)

== Installation ==

1. Upload `clustermaps` directory to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Add the user ID and site from Clustr Maps to the settings (Admin > Settings > Clustr Maps)
